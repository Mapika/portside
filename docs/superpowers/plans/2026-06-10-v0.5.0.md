### Task 2: ProxyJump (internal/sshconn)

**Files:** modify `config.go` (hop parsing), `conn.go` (chained dialing), tests.

- `Resolver.ProxyJump(alias) []string` ‚Äî split the `ProxyJump` value on commas, trim; empty or "none" ‚Üí nil.
- Hop literals: `parseHop(s string) (user, host, port string)` for `[user@]host[:port]`; if the hop matches a config Host alias, Resolve it through the config instead (literal user/port win if both present).
- Chained dial in `Connect`:

```go
func dialChain(r *Resolver, alias, secret string) (*ssh.Client, error) {
	hops := r.ProxyJump(alias)
	if len(hops) > 8 {
		return nil, errors.New("proxyjump chain too long")
	}
	var prev *ssh.Client
	for _, hop := range hops {
		p := r.resolveHop(hop)
		auth, closers := AuthMethods(p, secret)
		client, err := dialMaybeVia(prev, p.Addr, p.User, auth, hostKeyCallback())
		closeAll(closers)
		if err != nil {
			return nil, fmt.Errorf("via %s: %w", hop, err)
		}
		prev = client
	}
	p := r.Resolve(alias)
	auth, closers := AuthMethods(p, secret)
	client, err := dialMaybeVia(prev, p.Addr, p.User, auth, hostKeyCallback())
	closeAll(closers)
	return client, err
}

// dialMaybeVia dials directly when via == nil, else tunnels through via.
func dialMaybeVia(via *ssh.Client, addr, user string, auth []ssh.AuthMethod, hk ssh.HostKeyCallback) (*ssh.Client, error) {
	cfg := &ssh.ClientConfig{User: user, Auth: auth, HostKeyCallback: hk, Timeout: 10 * time.Second}
	if via == nil {
		return ssh.Dial("tcp", addr, cfg)
	}
	conn, err := via.Dial("tcp", addr)
	if err != nil {
		return nil, err
	}
	c, chans, reqs, err := ssh.NewClientConn(conn, addr, cfg)
	if err != nil {
		conn.Close()
		return nil, err
	}
	return ssh.NewClient(c, chans, reqs), nil
}
```

(Note: intermediate hop clients are deliberately NOT closed ‚Äî closing a hop kills the tunnel. Track them on Conn so Close() tears the whole chain down: add `hops []*ssh.Client` to Conn, closed in reverse order in Close(). Recursive alias ProxyJump (an alias hop having its own ProxyJump) is handled by resolving hops depth-first with a depth counter ‚Äî keep it simple and explicit.)

- `Connect(alias, secret)` uses dialChain, wraps SFTP as today.
- Tests: fixture config with `Host target / ProxyJump jump1` (+ a 3-hop chain case); start 2-3 in-process servers; assert Connect reaches the target and that Conn.Close closes cleanly. The in-process server allows direct-tcpip already.

Commit: `feat: proxyjump bastion support`
